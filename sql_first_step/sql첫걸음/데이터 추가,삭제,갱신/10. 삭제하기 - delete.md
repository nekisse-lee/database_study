## 삭제하기 - delete

```
delete from 테이블명 where 조건식;
```

### 1.  delete로 행 삭제하기

- 주의) delete * from 테이블명; 
   -  `where 구가 없는 경우 테이블의 전체 데이터가 삭제 됨으로 주의...`
- ex) 
   - select * from ttable;
   
   | no | a    | b          |
|:--:|:--:|:--:|
|  1 | abc  | 2019-01-01 |
|  2 | xyz  | NULL       |
|  3 | NULL | NULL       |

   - delete * from ttable where no=3;
     - no가 3인 데이터 삭제 
   
   - select * from ttable;
   
   | no | a    | b          |
|:--:|:--:|:--:|
|  1 | abc  | 2019-01-01 |
|  2 | xyz  | NULL       |
