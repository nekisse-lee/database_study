# 개체와 속성 (Entity and Attribute)

### 개체
- 실제 현실에서 독립적으로 존재하는 어떤 것
- 직사각형으로 표현

### 속성
- 개체를 설명할 수 있는 특성
- 타원으로 표현
- 키 속성 : 개체마다 고유한 값을 가지는 속성  
 밑줄로 표한하며 하나 이상 존재 가능
 

### Relationaship : 관계
- 개체와 개체 사이의 관계를 나타냄
- 관계도 애트리뷰트를 가질 수 있음

관계에는 1:1,  1:N, M:N  세 가지가 존재  
반드시 필요한 관계를 겹줄로 표현  
예) 사원 - 부서,   부서 - 매니저, 사원 - 프로젝
